# Paradise Nursery

Paradise Nursery is a React-based shopping cart web application designed for an online plant store.

## Description
This project allows users to browse different categories of plants, view details such as images, names, and prices, and add them to a shopping cart. Users can also manage cart items by increasing or decreasing quantities and viewing the total cost dynamically.

## Features
- Browse plants by categories
- Add plants to cart
- Update item quantities
- Remove items from cart
- Dynamic total price calculation

## Technologies Used
- React.js
- Redux Toolkit
- CSS
